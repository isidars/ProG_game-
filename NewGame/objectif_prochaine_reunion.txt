On demande le nom du joueur
le joueur entre son nom

on demande la classe du joueur
si le joueur tape n
    c'est une classe(n)
    l'equipier est donc la classe(n+x)

si le joueur tape n+x
    c'est une classe(n+x)
    l'equipier est donc la classe(n)
sinon
    on repose la question

1er combat
tant que le joueur ou le monstre on encore des points de vie le combat continue

    un monstre attaque
        le joueur perd des points de vie

    le joueur peut faire 3 actions
        Si il tape 1 il attaque le monstre
            alors le monstres perd de la vie
        Si il tape 2 il se soigne
            alors le joueur récupere des point de vie
        si il tente une autre action
            le joueur passe son tour

    L'equipier peux faire 2 action
        Si l'equipier choisi 1 il attaque le monstre
            le monstre perd des point de vie
        Si l'equipier choisi 2 il soigne le joueur
            Le joueur est soigner de quelque point de vie

    Si le joueur perd tous ces points de vie
        Le combat est fini et le joueur à perdu
        on demande au joueur si il veut réessayer
        si il dit oui
            Le joueur et le monstre recupere leur point de vie
            le combat recommence
        Si il dit non
            Le jeux est fini.

    Si le monstre perd tous ces points de vie
        Le combat est fini le joueurs est recompensé par du loot et de l'or.
